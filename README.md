# Consultório Web

Sistema de gestão para consultório de psicologia: agenda de consultas,
fluxo financeiro, lembretes via WhatsApp e relatórios.

## Tecnologias
- Java 21 + Spring Boot 3.x
- Spring Data JPA
- H2 (dev) / PostgreSQL (prod)
- Maven
